# O que foi usado nos projetos

### Backend
  - Node
  - Express
  - Javascript
  - Cors

### Frontend
  - React
  - Javascript


### Como eu faço para executar o serviço do backend?

```sh
$ cd server
$ npm install
$ npm start
```

Feito isso o serviço de backend estará funcionando na porta :5000. 

### Como eu faço para executar o serviço do frontend?

```sh
$ cd web
$ npm install
$ npm start
```
